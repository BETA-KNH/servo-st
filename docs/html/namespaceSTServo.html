Detailed Description

Serial port parameters needed to open the servo bus connection.

Parsed representation of a response packet sent by the servo.

Populated from environment variables so no recompile is needed when targeting a different device or baud rate.

Required environment variables (or entries in a servo.env file):

  • SERVO_PORT — path to the serial device
  • SERVO_BAUD — baud rate integer

Optional:

  • SERVO_TIMEOUT_MS — read timeout in ms (default: 100)

Env-file search order (first match per key wins, shell env always wins):

  1. servo.env (CWD)
  2. conf/servo.env (CWD/conf — development layout)
  3. /etc/servo_sensei/servo.env (system-wide install on embedded targets)

Packet layout:

[0xFF] [0xFF] [ID] [LEN] [ERROR] [DATA...] [CHECKSUM]
LEN = sizeof(ERROR) + sizeof(DATA) + sizeof(CHECKSUM)
CHECKSUM = ~(ID + LEN + ERROR + DATA...) & 0xFF
